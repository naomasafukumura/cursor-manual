# 1. GitHubに登録する

**GitHub（ギットハブ）** は、チームのファイルを保管する場所です。
ここにファイルを置くことで、みんなで安全に共同編集できます。

::: warning もうGitHubアカウントを持っている人
ここを飛ばして次のステップに進んでください。
:::

## アカウントを作る

1. ブラウザで **[github.com](https://github.com)** を開く
2. 右上の **「Sign up」** をクリック
3. **メールアドレス** を入力
4. **パスワード** を決めて入力（8文字以上）
5. **ユーザー名** を決めて入力（英語で。例：`masa-taro`）
6. メールに届いた **認証コード** を入力
7. 完了！

## Personal Access Token を作る（Macの人だけ）

ファイルをダウンロード・アップロードするときに使う「パスワードの代わり」を作ります。

**Windowsの人はこの手順は不要です。** 次のステップに進んでください。

1. ブラウザで **[github.com/settings/tokens](https://github.com/settings/tokens)** を開く
2. 左のメニューで **「Tokens (classic)」** をクリック
3. **「Generate new token ▼」** をクリック
4. 下に出てくる **「Generate new token (classic)」** をクリック（上のではなく **下の方** です）
5. 以下を入力：
   - 「Note」に **`cursor-manual`** と入力
   - 「Expiration」を **「No expiration」** に変更
   - 「repo」に **チェック** を入れる
6. 一番下の **「Generate token」** をクリック
7. `ghp_` で始まる文字列が表示される → **コピーしてメモ帳などに保存**

::: danger この文字列は超大事！
表示されるのは **1回だけ** です。必ずコピーして安全な場所に保存してください。
後でファイルをダウンロードするときに使います。
:::
