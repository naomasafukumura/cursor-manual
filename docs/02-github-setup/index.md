# 2. GitHubアカウント作成 & 招待を受ける

GitHub（ギットハブ）は、ファイルをチームで共有・管理するためのサービスです。
ここでは、アカウントを作成して Masa English の組織に参加する手順を説明します。

## GitHubアカウントを作成する

::: warning すでにアカウントをお持ちの方
GitHubアカウントをすでにお持ちの場合は、この手順をスキップして「[招待を受け入れる](#招待を受け入れる)」に進んでください。
:::

### 手順

1. **GitHubにアクセス**
   ブラウザで [https://github.com](https://github.com) を開きます

2. **「Sign up」をクリック**
   右上の「Sign up」ボタンをクリックします

   ::: tip スクリーンショット
   📸 ここにGitHubトップページのスクリーンショットが入ります
   :::

3. **メールアドレスを入力**
   会社のメールアドレス（`xxx@masaenglishcompany.com`）を入力します

4. **パスワードを設定**
   8文字以上で、数字と小文字を含むパスワードを設定します

5. **ユーザー名を設定**
   ユーザー名を入力します（例：`masa-taro`）
   ※ 他の人と被らない名前にしてください

6. **メール認証**
   登録したメールアドレスに認証コードが届くので、入力します

7. **アカウント作成完了！**

## 招待を受け入れる

Masa English の組織（Organization）への招待メールが届いているはずです。

### 手順

1. **メールを確認**
   GitHubから「You've been invited to join the **Masa-English** organization」というメールが届いています

   ::: tip スクリーンショット
   📸 ここに招待メールのスクリーンショットが入ります
   :::

2. **「Join Masa-English」をクリック**
   メール内のボタンをクリックします

3. **招待を受け入れる**
   GitHubのページが開くので、**「Join Masa-English」** をクリックします

::: info 招待メールが届かない場合
- 迷惑メールフォルダを確認してください
- それでも届かない場合は、管理者に連絡してください
:::

## GitHubにサインインした状態を維持する

ブラウザでGitHubにサインインした状態にしておくと、後の手順がスムーズです。
