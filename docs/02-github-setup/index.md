# 2. GitHubに登録する

**GitHub（ギットハブ）** は、ファイルをインターネット上で共有するサービスです。
Google ドライブのチーム版みたいなものだと思ってください。

## アカウントを作る

::: warning もうアカウントを持っている人
GitHubアカウントがある人は、ここを飛ばして「[チームに参加する](#チームに参加する)」へ進んでください。
:::

1. ブラウザで **[github.com](https://github.com)** を開く
2. 右上の **「Sign up」** をクリック
3. **メールアドレス**を入力（会社のメールアドレス）
4. **パスワード**を決めて入力（8文字以上）
5. **ユーザー名**を決めて入力（例：`masa-taro`、英語で）
6. メールに届いた**認証コード**を入力
7. 完了！

## チームに参加する

管理者から招待メールが届いているはずです。

1. メールの受信トレイを見る（「**Masa-English**」で検索）
2. メールの中の **「Join Masa-English」** ボタンをクリック
3. 開いたページで **「Join Masa-English」** をクリック

::: info メールが届いてない？
迷惑メールフォルダを確認してください。それでもなければ管理者に連絡してください。
:::
