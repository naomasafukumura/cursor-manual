# 2. GitHubに登録する

**GitHub（ギットハブ）** は、チームのファイルを保管する場所です。
ここにファイルを置くことで、みんなで安全に共同編集できます。

## アカウントを作る

::: warning もうアカウントを持っている人
ここを飛ばして「[チームに参加する](#チームに参加する)」へ進んでください。
:::

1. ブラウザで **[github.com](https://github.com)** を開く
2. 右上の **「Sign up」** をクリック
3. **メールアドレス** を入力
4. **パスワード** を決めて入力（8文字以上）
5. **ユーザー名** を決めて入力（英語で。例：`masa-taro`）
6. メールに届いた **認証コード** を入力
7. 完了！

## チームに参加する

管理者から招待メールが届きます。

1. メールで「**Masa-English**」を検索
2. **「Join Masa-English」** をクリック
3. 開いたページで **「Join Masa-English」** をクリック

::: info メールが届かない？
迷惑メールフォルダを確認してください。それでもなければ管理者に連絡。
:::

## Personal Access Token を作る（Macの人だけ）

ファイルをダウンロード/アップロードするときに使う「カギ」を作ります。

1. ブラウザで **[github.com/settings/tokens](https://github.com/settings/tokens)** を開く
2. 左メニューの **「Tokens (classic)」** をクリック
3. **「Generate new token」** → **「Generate new token (classic)」** をクリック
4. 「Note」に **`Cursor`** と入力
5. 「Expiration」を **「No expiration」** に変更
6. **「repo」** にチェック ✅
7. 一番下の **「Generate token」** をクリック
8. `ghp_` で始まる文字列を **コピーしてメモ帳に保存**

::: danger 表示されるのは1回だけ！
必ずコピーして保存してください。これがパスワードの代わりになります。
:::

**Windowsの人** はこの手順は不要です。ダウンロード時にブラウザでログインするだけでOK。
