# 3. Claude Codeを入れる

**Claude Code（クロード・コード）** は、Cursorの中で使えるAIアシスタントです。
日本語で「〇〇して」と頼むだけで、ファイルの編集やGitの操作を全部やってくれます。

## Node.js を入れる（まだの人だけ）

Claude Codeを入れるには、まず「Node.js」が必要です。

1. ブラウザで **[nodejs.org](https://nodejs.org/)** を開く
2. **「LTS」** と書いてある方のボタンをクリックしてダウンロード
3. ダウンロードされたファイルを開いてインストール（全部「Next」でOK）

## Claude Code をインストールする

1. **Cursorを開く**
2. キーボードの **`Ctrl + `` `** を押す（Ctrlキーを押しながら、左上のバッククォートキー `` ` ``）
3. 画面の下に黒い画面（ターミナル）が出てきます
4. そこに以下をコピー＆ペーストして **Enter** を押す：

```bash
npm install -g @anthropic-ai/claude-code
```

5. いろいろ文字が流れて、最後にエラーが出なければOK

## Claude Code にログインする

1. ターミナルに以下を打って **Enter**：

```bash
claude
```

2. 画面にログイン方法の選択が出ます → **「Google アカウントでログイン」** を選ぶ
3. ブラウザが自動で開きます
4. **Googleアカウント**（会社から指定されたアカウント）でサインイン
5. 「許可しますか？」と聞かれたら → **「許可」** をクリック
6. ブラウザに「認証完了」と表示されたら、**Cursorのターミナルに戻る**
7. ターミナルに `claude >` と表示されていればログイン成功！

::: info ログインは1回だけ
一度ログインすれば、次からは `claude` と打つだけで使えます。
:::

## 動作確認

ターミナルに以下を打ってみてください：

```
こんにちは
```

Claude が日本語で返事をしてくれたら成功です！

Claude Code を終わるときは `/exit` と打ってください。
