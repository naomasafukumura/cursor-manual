# 3. Claude Codeを入れる

**Claude Code（クロード・コード）** は、Cursorの中で使えるAIアシスタントです。
日本語で「〇〇して」と頼むだけで、ファイルの編集やGitの操作を全部やってくれます。

## Node.js を入れる（まだの人だけ）

Claude Codeを入れるには、まず「Node.js」が必要です。

### Macの人

1. ブラウザで **[nodejs.org](https://nodejs.org/)** を開く
2. **「Node.jsを入手」** ボタンをクリック
3. ダウンロードページが開くので、**「macOS インストーラー (.pkg)」** をクリック
4. ダウンロードされたファイルを開いてインストール（全部「続ける」でOK）

### Windowsの人

1. ブラウザで **[nodejs.org](https://nodejs.org/)** を開く
2. **「Node.jsを入手」** ボタンをクリック
3. ダウンロードページが開くので、**「Windows インストーラー (.msi)」** をクリック
4. ダウンロードされたファイルを開いてインストール（全部「Next」でOK）

## Claude Code をインストールする

1. **Cursorを開く**
2. スタート画面で **「Open project」** をクリック → 適当なフォルダを選んで **「開く」**（どのフォルダでもOK）
3. 画面上のメニューから **「Terminal」→「New Terminal」** をクリック

![ターミナルの開き方](/images/cursor-terminal.svg)

4. 画面の下に黒い画面（ターミナル）が出てきます
5. そこに以下をコピー＆ペーストして **Enter** を押す：

```bash
npm install -g @anthropic-ai/claude-code
```

6. いろいろ文字が流れて、最後にエラーが出なければOK

## Claude Code 拡張機能を入れる（任意）

Cursorのマーケットプレイスから**Claude Code拡張機能**を入れると、ターミナルを開かなくてもClaude Codeが使えるようになります。

1. Cursorの左側にある **拡張機能アイコン**（四角が4つのマーク）をクリック
2. 検索欄に **「Claude」** と入力
3. **「Claude Code for VS Code」**（Anthropic製）が表示される → **「インストール」** をクリック

::: info ターミナル版と拡張機能版の違い
どちらも同じClaude Codeです。好きな方を使ってください。
- **ターミナル版**（`claude` コマンド）→ Cursorが使えないときでも動く
- **拡張機能版** → Cursorの中でボタンから起動できて手軽
:::

::: warning Macで「permission denied」エラーが出た場合
以下のようなエラーが出ることがあります：

```
npm error Error: EACCES: permission denied
```

**方法①（かんたん）：** 先頭に `sudo` をつけて再実行

```bash
sudo npm install -g @anthropic-ai/claude-code
```

パスワードを聞かれたら、**Macのログインパスワード**を入力して Enter を押してください（入力中は画面に何も表示されませんが、正常です）。

**方法②（sudoでもダメな場合）：** npmの保存先を変更

以下の3行をターミナルにコピー＆ペーストして実行してください：

```bash
mkdir -p ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH="$HOME/.npm-global/bin:$PATH"' >> ~/.zshrc
```

実行したら、**ターミナルを一度閉じて開き直してから**、もう一度インストールしてください：

```bash
npm install -g @anthropic-ai/claude-code
```
:::

## Claude Code にログインする

1. ターミナルに以下を打って **Enter**：

```bash
claude
```

2. 画面にログイン方法の選択が出ます → **「Google アカウントでログイン」** を選ぶ
3. ブラウザが自動で開きます

::: tip ブラウザが自動で開かない場合
ターミナルに長いURLが表示されます。そのURLをコピーしてブラウザに貼り付けてください。
:::

4. ブラウザに **認証画面** が表示されます → **Googleアカウント**（会社から指定されたアカウント）でサインイン
5. 「許可しますか？」と聞かれたら → **「許可」** をクリック
6. ブラウザに「認証完了」と表示されたら、**Cursorのターミナルに戻る**
7. ターミナルに `claude >` と表示されていればログイン成功！

::: danger ここで claude.ai にアクセスしないでください！
ログイン中にブラウザで **claude.ai のチャット画面** が開いてしまった場合、それは Claude Code のログインではありません。

Claude Code のログインは **認証の許可画面** です。claude.ai でチャットを始めてしまった場合は、そのタブを閉じて、ターミナルに表示されたURLを改めてブラウザに貼り付けてください。
:::

::: info ログインは1回だけ
一度ログインすれば、次からは `claude` と打つだけで使えます。
:::

## 動作確認

ターミナルに以下を打ってみてください：

```
こんにちは
```

Claude が日本語で返事をしてくれたら成功です！

Claude Code を終わるときは `/exit` と打ってください。

## よくあるトラブル

### Cursorで変なファイルが開いている

Cursorを開いたとき、以下のような画面になっていたら、**開くフォルダが間違っています**：

- `ReleaseNotes.html` や `Program Files` の中のファイルが表示されている
- 見覚えのないHTMLやコードが表示されている

**対処法：**
1. Cursorの左上 **「File」→「Open Folder」** をクリック
2. [ステップ5](/05-clone/) でダウンロードしたプロジェクトフォルダを選び直してください

### `npm: command not found` と出る

Node.js がインストールされていません。このページの最初にある **「Node.js を入れる」** の手順をやってください。

### `permission denied` と出る（Mac）

上の「Claude Code をインストールする」にある方法①または方法②を試してください。

### Node.jsを入れたのに `npm: command not found` と出る

Node.jsをインストールした直後は、ターミナルがまだ認識していないことがあります。

**対処法：** ターミナルを一度閉じて、もう一度開き直してください。

### `xcode-select: note: No developer tools were found` と出る（Mac）

Gitが使えない状態です。[ステップ1の「Gitを入れる」](/01-cursor-install/#gitを入れる) の Mac の手順を実行してください。

### Cursorで「複数端末エラー」が出て操作できない

Cursorのライセンスは同時に1台しか使えないため、別のパソコンでCursorを開いていると**エラーで操作できなくなる**ことがあります。

**対処法：**
- 他のパソコンでCursorを閉じる
- それができない場合は、**Claude Codeをターミナルから直接使えばCursorなしでも作業できます**

ターミナル（Mac：ターミナル.app、Windows：PowerShell）を開いて：

```bash
cd プロジェクトのフォルダ
claude --dangerously-skip-permissions
```

あとは普段通り日本語で指示すればOKです。

### エラーが大量に出てパニックになった

::: danger エラーメッセージをターミナルにコピー＆ペーストしないでください！
エラーの文字列がそのまま「コマンド」として実行されてしまい、エラーがさらに増えて悪循環になります。
実際にこれでハマった事例があります。
:::

**対処法：**
1. まず **`Ctrl + C`** を何回か押してターミナルを落ち着かせる
2. エラー画面の **スクリーンショット** を撮る
3. Slackで相談してください

## 困ったときの緊急ボタン

操作中に「おかしいな？」と思ったら、以下の2つを覚えておいてください。

| 状況 | やること | ボタン |
|------|---------|--------|
| Claude が長時間動いていて終わらない | 今の処理を中断する | **`Esc`** キーを押す |
| ターミナルが変な状態になった | ターミナルを強制リセットする | **`Ctrl + C`** を2回押す |

::: tip まずは `Esc`、ダメなら `Ctrl + C`
どちらを押しても壊れることはありません。安心して押してください。
:::
