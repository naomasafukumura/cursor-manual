# 3. Claude Codeにログインする

**Claude Code（クロード・コード）** は、Cursorのターミナルで使えるAIアシスタントです。
日本語で「〇〇して」と頼むだけで、ファイルの編集やGitの操作を全部やってくれます。

## ログインする

1. Cursorを開いて、**Terminal → New Terminal** でターミナルを表示
2. 以下を打って **Enter**：

```bash
claude
```

3. 画面にログイン方法の選択が出ます → **「Google アカウントでログイン」** を選ぶ
4. ブラウザが自動で開きます

::: tip ブラウザが自動で開かない場合
ターミナルに長いURLが表示されます。そのURLをコピーしてブラウザに貼り付けてください。
:::

5. ブラウザに **認証画面** が表示されます → **Googleアカウント**（会社から指定されたアカウント）でサインイン
6. 「許可しますか？」と聞かれたら → **「許可」** をクリック
7. ブラウザに「認証完了」と表示されたら、**ターミナルに戻る**
8. ターミナルに `claude >` と表示されていればログイン成功！

::: danger ここで claude.ai にアクセスしないでください！
ログイン中にブラウザで **claude.ai のチャット画面** が開いてしまった場合、それは Claude Code のログインではありません。

Claude Code のログインは **認証の許可画面** です。claude.ai でチャットを始めてしまった場合は、そのタブを閉じて、ターミナルに表示されたURLを改めてブラウザに貼り付けてください。
:::

::: info ログインは1回だけ
一度ログインすれば、次からは `claude` と打つだけで使えます。
:::

## 動作確認

ターミナルに以下を打ってみてください：

```
こんにちは
```

Claude が日本語で返事をしてくれたら成功です！

Claude Code を終わるときは `/exit` と打ってください。
