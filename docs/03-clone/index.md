# 3. ファイルをダウンロードする

GitHubにあるチームのファイルを、自分のパソコンにダウンロードします。
これを「**クローン**」と呼びます。**最初に1回だけ** やればOKです。

## 手順

1. Cursorを開くと、スタート画面が表示されます

2. **「Clone repo」** をクリック

   ::: info どこにあるの？
   スタート画面の上部に「Open project」「Clone repo」「Connect via SSH」の3つのボタンがあります。真ん中の **「Clone repo」** です。
   :::

3. 上に出てくる入力欄に、**リポジトリのURLを貼り付け** ます

   例：`https://github.com/Masa-English/kpi-dashboard.git`

   ::: tip URLはどこにある？
   GitHubでリポジトリのページを開いて、緑色の **「Code」** ボタンをクリック → **「HTTPS」** のURLをコピーします。
   :::

4. **保存先を選ぶ画面** が出るので、「Documents」フォルダを選んで **「OK」**

5. **ログインを求められたら：**
   - **Mac** → ユーザー名を入れて、パスワード欄には **Token**（`ghp_`で始まる文字列）を貼り付け
   - **Windows** → ブラウザが開くのでGitHubにサインイン

6. ダウンロードが終わると、Cursorでファイルが開きます

7. 「Trust the authors?」と出たら → **「Yes, I trust the authors」**

左側にファイルの一覧が出ればOK！
