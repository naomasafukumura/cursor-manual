# 4. リポジトリをクローンする

「クローン」とは、GitHub上にあるファイルを自分のパソコンにダウンロードすることです。
これにより、チームの最新ファイルを自分のパソコンで編集できるようになります。

## リポジトリのURLをコピーする

### 手順

1. **GitHubにアクセス**
   ブラウザで Masa-English のリポジトリページを開きます
   例：`https://github.com/Masa-English/リポジトリ名`

2. **「Code」ボタンをクリック**
   緑色の「Code」ボタンをクリックします

3. **URLをコピー**
   「HTTPS」タブが選択されていることを確認して、URLの横にある 📋 ボタンをクリックしてコピーします

   ::: info UIの場所
   「Code」ボタンはリポジトリページのファイル一覧の上、右側にある緑色のボタンです。クリックするとHTTPSのURLが表示されます。
   :::

## Cursorでクローンする

### 手順

1. **Cursorを起動**します

2. **ターミナルを開く**
   - Mac：**`Ctrl + `` `**
   - Windows：**`Ctrl + `` `**

3. **保存先のフォルダに移動**
   ファイルを保存したい場所に移動します。おすすめはホームディレクトリの下です：

   **Mac の場合：**
   ```bash
   cd ~/Documents
   ```

   **Windows の場合：**
   ```bash
   cd C:\Users\あなたのユーザー名\Documents
   ```

4. **クローンコマンドを実行**
   以下のコマンドを入力して Enter を押します：

   ```bash
   git clone https://github.com/Masa-English/リポジトリ名.git
   ```

5. **認証を求められた場合**
   - **Mac**：ユーザー名と Personal Access Token を入力
   - **Windows**：ブラウザが開くので GitHub にサインイン

6. **クローン完了！**
   `Cloning into 'リポジトリ名'... done.` と表示されれば成功です

## クローンしたフォルダをCursorで開く

### 手順

1. **Cursorのメニュー** → **「File」** → **「Open Folder...」**

2. クローンしたフォルダ（`リポジトリ名`）を選択して **「Open」**

3. **「Trust the authors」** と聞かれたら → **「Yes, I trust the authors」** をクリック

左側のサイドバーにファイル一覧が表示されれば成功です！🎉
