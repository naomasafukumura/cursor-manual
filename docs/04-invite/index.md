# 4. 管理者からの招待を受ける

チームのファイルにアクセスするには、管理者からの **招待** が必要です。

## メンバー向け：招待を受ける

1. 管理者が招待を送ると、GitHubに登録したメールアドレスに **招待メール** が届きます
2. メールの中の **「Join Masa-English」** ボタンをクリック
3. 開いたページで **「Join Masa-English」** をクリック
4. これでチームのファイルにアクセスできるようになります！

::: info メールが届かない？
- 迷惑メールフォルダを確認してください
- 「Masa-English」や「GitHub」で検索してみてください
- それでもなければ管理者に連絡してください
:::

---

## 管理者向け：メンバーを招待する方法

::: warning このセクションは管理者向けです
一般メンバーはここを飛ばして次のステップに進んでください。
:::

### Organization に招待する

1. ブラウザで **[github.com/orgs/Masa-English/people](https://github.com/orgs/Masa-English/people)** を開く
2. **「Invite member」** をクリック
3. メンバーの **GitHubユーザー名** または **メールアドレス** を入力
4. **「Invite」** をクリック

これでメンバーに招待メールが届きます。

### 特定のリポジトリだけに招待する場合

Organization全体ではなく、特定のリポジトリだけ共有したい場合：

1. リポジトリのページを開く（例：`https://github.com/Masa-English/kpi-dashboard`）
2. **「Settings」** タブ → **「Collaborators and teams」**
3. **「Add people」** → メンバーを追加
