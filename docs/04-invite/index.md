# 4. 招待を受ける

チームのファイルにアクセスするには、管理者からの **招待** が必要です。

## メンバー向け：招待を受ける

### GitHub の招待

1. 管理者が招待を送ると、GitHubに登録したメールアドレスに **招待メール** が届きます
2. メールの中の **「Join Masa-English」** ボタンをクリック
3. 開いたページで **「Join Masa-English」** をクリック
4. これでチームのファイルにアクセスできるようになります！

### Vercel の招待

プロジェクトによっては、APIキーなどの設定情報を取得するために **Vercelのチームにも参加** する必要があります。

1. 管理者からVercelの招待メールが届きます（**「Vercel」** で検索してみてください）
2. メールの中の **招待リンク** をクリック
3. Vercelにログイン（アカウントがなければ作成）→ チームに参加

::: info Vercelの招待が届かない場合
管理者に「Vercelのチームにも招待してください」と連絡してください。
:::

::: info メールが届かない？
- 迷惑メールフォルダを確認してください
- 「Masa-English」「GitHub」「Vercel」で検索してみてください
- それでもなければ管理者に連絡してください
:::

## 管理者向け：メンバーを招待する方法

::: warning このセクションは管理者向けです
一般メンバーはここを飛ばして次のステップに進んでください。
:::

### Organization に招待する

1. ブラウザで **[github.com/orgs/Masa-English/people](https://github.com/orgs/Masa-English/people)** を開く
2. **「Invite member」** をクリック
3. メンバーの **GitHubユーザー名** または **メールアドレス** を入力
4. **「Invite」** をクリック

これでメンバーに招待メールが届きます。

### 特定のリポジトリだけに招待する場合

Organization全体ではなく、特定のリポジトリだけ共有したい場合：

1. リポジトリのページを開く（例：`https://github.com/Masa-English/kpi-dashboard`）
2. **「Settings」** タブ → **「Collaborators and teams」**
3. **「Add people」** → メンバーを追加

### Vercel にも招待する

APIキーなどの環境変数を共有するには、**Vercelのチームにも招待が必要** です。
これをしないと、メンバーがローカル開発で環境変数を取得できません。

1. ブラウザで **[vercel.com](https://vercel.com)** にログイン
2. 左上のチーム名（**MasaEnglish**）をクリック
3. **「Settings」** → **「Members」**
4. **「Invite Member」** をクリック
5. メンバーの **メールアドレス** を入力して招待

::: tip GitHub と Vercel の両方に招待してください
- **GitHub** → コード（ファイル）にアクセスできるようになる
- **Vercel** → APIキーなどの環境変数・デプロイ情報にアクセスできるようになる

どちらか片方だけだと、メンバーがローカルでアプリを動かせなかったり、デプロイURLが確認できなかったりします。
:::

::: warning Vercelにプロジェクトをデプロイ済みであることを確認
メンバーが `vercel link` でプロジェクトを紐づけるには、管理者側でそのリポジトリがVercelにデプロイされている必要があります。
まだの場合は、先にVercelのダッシュボードからプロジェクトを作成・デプロイしてください。
:::
