# 4. 編集のやり方（Claude Codeを使う）

ここからが毎回の作業です。**全部 Claude Code に日本語で話しかけるだけ** です。

## Claude Code を起動する

1. Cursorでプロジェクトを開く（Recent projectsからクリック）
2. **`Ctrl + `` `** でターミナルを開く
3. 以下を打って Enter：
   ```bash
   claude
   ```

## 毎回の流れ

### ① まず最新にする

```
「更新ある？あったら最新にして」
```

他のメンバーが変更していたら、自動で取ってきてくれます。
**作業を始める前に必ずやってください。**

### ② ブランチを作る（大きな変更のとき）

```
「ブランチ作って。名前は feature/update-dashboard で」
```

::: info ブランチって何？
**下書きノート** のようなものです。本番（黒板）を壊さずに、安全に作業できます。
リンクを1個変える程度ならブランチなしでもOKです。
:::

### ③ ファイルを編集する

```
「DASHBOARD.html のタイトルを『KPI分析』に変えて」
「AnalysisForm.html の送信先URLを変更して」
「新しいページを追加して」
```

日本語で具体的に指示するだけ。Claude が編集してくれます。

::: warning 確認してから承認
Claude が「変更していいですか？」と聞いてきたら、**内容を見てから** 「y」を押してください。
よくわからなければ「n」でキャンセルできます。
:::

### ④ 変更を送る

```
「コミットしてプッシュして」
```

これでGitHubにアップロードされます。

### ⑤ レビューを依頼する（ブランチで作業した場合）

```
「プルリクエスト作って」
```

他のメンバーに「確認してください」と依頼できます。
確認がOKなら、本番に反映されます。

## まとめ

| やること | Claude に言うこと |
|---------|-----------------|
| 最新にする | 「更新ある？」 |
| 下書き作る | 「ブランチ作って」 |
| 編集する | 「ここを直して」 |
| 送る | 「コミットしてプッシュして」 |
| 確認依頼 | 「プルリクエスト作って」 |

**この5つだけ覚えればOKです！**

## Claude Code を終わるとき

- `/exit` と打つ
- または **`Ctrl + C`** を2回押す

→ わからない言葉は [用語集](/glossary) を見てください
