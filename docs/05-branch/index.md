# 5. 作業スペースを作る

ファイルを編集する前に、**自分だけの作業スペース**（ブランチ）を作ります。

## なぜ作業スペースが必要？

例えるなら：
- チームの **本番ファイル** = 教室の黒板
- **作業スペース** = 自分のノート

まず自分のノート（作業スペース）に下書きして、完成したら黒板（本番）に書き写す、というイメージです。こうすれば、途中の下書きで黒板を汚さずにすみます。

## 作業スペースを作る手順

1. Cursorの **左下** を見る → 「**main**」と書いてある部分をクリック

   ::: info どこにあるの？
   Cursorの一番下にあるバーの左端に、枝分かれマークと「main」という文字があります。
   :::

2. 上に出てくるメニューの **「+ Create new branch...」** をクリック
3. 作業スペースの名前を入力して Enter

## 名前の付け方

| やること | 名前の例 |
|---------|---------|
| 何かを追加する | `feature/add-lesson-10` |
| 間違いを直す | `fix/typo-in-lesson-5` |
| 説明文を更新する | `docs/update-readme` |

::: warning ルール
- **英語の小文字** で書く
- スペースは使わず **ハイフン `-`** を使う
- 例：`feature/add-new-file` ← こんな感じ
:::

## 確認

Cursorの左下が **「main」以外の名前** に変わっていればOKです！
