# 6. ファイルを編集する

いよいよファイルを編集します。普通のテキスト編集と同じです。

## ファイルを開く

1. Cursorの **左側** にファイル一覧がある
2. 編集したいファイルを **クリック**
3. 右側に内容が表示される

::: tip すばやくファイルを探す
**`Cmd + P`**（Mac）/ **`Ctrl + P`**（Windows）を押すと、ファイル名で検索できます。
:::

## 編集する

WordやGoogleドキュメントと同じです。

- クリックした場所にカーソルが移動
- 文字を打てば入力される
- **`Cmd + C`** / **`Ctrl + C`** → コピー
- **`Cmd + V`** / **`Ctrl + V`** → 貼り付け

## 保存する

- **Mac：`Cmd + S`**
- **Windows：`Ctrl + S`**

::: tip 自動保存をオンにすると楽です
1. **`Cmd + ,`**（Mac）/ **`Ctrl + ,`**（Windows）で設定を開く
2. 検索欄に **`Auto Save`** と入力
3. **「afterDelay」** に変更

これで勝手に保存されるようになります。
:::

## 編集したファイルの見分け方

ファイル名のタブに **白い丸 ●** がついていたら → まだ保存されていない
保存すると ● が消えます。
