# 6. ファイルを編集・保存する

ブランチを作成したら、いよいよファイルを編集します。
ここでは、Cursorでファイルを開いて編集・保存する方法を説明します。

## ファイルを開く

### 手順

1. **左側のサイドバー（エクスプローラー）** でファイル一覧を確認します

   ::: tip スクリーンショット
   📸 ここにサイドバーのスクリーンショットが入ります
   :::

2. **編集したいファイルをクリック**
   右側のエディタ領域にファイルの内容が表示されます

3. ファイルを探すときは **`Cmd + P`**（Mac）/ **`Ctrl + P`**（Windows）で
   ファイル名を入力して素早く開くこともできます

## ファイルを編集する

ファイルが開いたら、普通のテキストエディタと同じように編集できます。

- クリックしてカーソルを移動
- テキストを入力・削除
- コピー（`Cmd + C` / `Ctrl + C`）、ペースト（`Cmd + V` / `Ctrl + V`）

### 編集中の目印

ファイルを変更すると、タブのファイル名の横に **白い丸 ●** が表示されます。
これは「保存されていない変更がある」という意味です。

::: tip スクリーンショット
📸 ここに未保存マークのスクリーンショットが入ります
:::

## ファイルを保存する

### 手順

- **Mac：** **`Cmd + S`**
- **Windows：** **`Ctrl + S`**

保存すると、タブのファイル名の横の **白い丸 ●** が消えます。

::: info 自動保存を有効にする（おすすめ）
Cursorの設定で自動保存を有効にすると、編集のたびに自動で保存されます：

1. **`Cmd + ,`**（Mac）/ **`Ctrl + ,`**（Windows）で設定を開く
2. 検索欄に **`Auto Save`** と入力
3. **「Files: Auto Save」** を **「afterDelay」** に変更

📸 ここに自動保存設定のスクリーンショットが入ります
:::

## Markdownファイルの場合

`.md` ファイル（Markdownファイル）を編集する場合、プレビュー機能が便利です。

1. `.md` ファイルを開く
2. 右上の **プレビューアイコン** をクリック（目のようなマーク）
3. 右側にプレビューが表示されます

::: tip スクリーンショット
📸 ここにMarkdownプレビューのスクリーンショットが入ります
:::
