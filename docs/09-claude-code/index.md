# 9. AIに手伝ってもらう

**Claude Code（クロード・コード）** は、ターミナルで使えるAIアシスタントです。
日本語で「〇〇して」と頼むだけで、ファイルの編集を手伝ってくれます。

## インストールする

1. Cursorでターミナルを開く（**`Ctrl + `` `**）
2. 以下を打って Enter：
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

   ::: warning 「npm が見つかりません」と出たら
   先に **Node.js** を入れる必要があります。
   [nodejs.org](https://nodejs.org/) から「LTS」版をダウンロードしてインストールしてください。
   :::

3. 入ったか確認：
   ```bash
   claude --version
   ```
   数字が出ればOK。

## ログインする

1. ターミナルに `claude` と打って Enter
2. 「Google アカウントでログイン」を選ぶ
3. ブラウザが開くので、Googleアカウントでサインイン
4. ターミナルに戻ればOK

## 使い方

普通の日本語で話しかけるだけです。

**例：**

| やりたいこと | こう打つ |
|-------------|---------|
| ファイルの中身を見たい | `このフォルダの中身を教えて` |
| ファイルを直したい | `lesson-5.md の3番目の例文を修正して` |
| 新しいファイルを作りたい | `新しいレッスンファイルを作って` |
| 間違いを見つけたい | `このファイルの誤字を見つけて` |

::: warning 大事なこと
Claude Code が「変更していいですか？」と聞いてきたら、**内容を確認してから**「y」（はい）を押してください。
よくわからないときは「n」（いいえ）を押せばキャンセルできます。
:::

## 終わるとき

- `/exit` と打つ
- または **`Ctrl + C`** を2回押す
