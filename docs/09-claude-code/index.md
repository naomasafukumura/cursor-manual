# 9. Claude Codeの使い方

Claude Code（クロード・コード）は、ターミナルで使えるAIアシスタントです。
Cursorのターミナルから直接使うことで、作業を大幅に効率化できます。

## Claude Codeとは？

- ターミナルに質問や指示を入力するだけで、AIがファイルの編集を手伝ってくれます
- ファイルの内容を理解して、適切な修正を提案・実行してくれます
- コードだけでなく、Markdownファイル（ドキュメント）の編集にも使えます

## インストール

### 手順

1. **Cursorのターミナルを開く**
   - Mac：**`Ctrl + `` `**
   - Windows：**`Ctrl + `` `**

2. **Claude Codeをインストール**
   以下のコマンドを実行します：

   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

   ::: warning Node.jsが必要です
   `npm` コマンドが見つからない場合は、まず [Node.js](https://nodejs.org/) をインストールしてください。
   公式サイトから「LTS」版をダウンロードしてインストールします。
   :::

3. **インストール完了の確認**
   ```bash
   claude --version
   ```
   バージョン番号が表示されればOKです。

## ログイン

Claude Codeを使うには、最初にログインが必要です。

### 手順

1. **ターミナルで以下を実行：**
   ```bash
   claude
   ```

2. **ログイン方法を選択**
   「Google アカウントでログイン」を選択します

3. **ブラウザが開く**
   Googleアカウントでサインインします

4. **認証完了**
   ターミナルに戻ると、Claude Codeが使えるようになっています

## 基本的な使い方

Claude Code を起動した状態で、自然な日本語で指示を入力します。

### よく使うコマンド例

| やりたいこと | 入力例 |
|-------------|--------|
| ファイルの内容を確認 | `このフォルダの中身を教えて` |
| ファイルを編集 | `lesson-5.md の3番目の例文を修正して` |
| 新しいファイルを作成 | `新しいレッスンファイルを作成して` |
| 誤字を修正 | `このファイルの誤字を見つけて修正して` |
| 内容を要約 | `README.md の内容を要約して` |

### 使用例

```
あなた: lesson-5.md の中の「英語」を「English」に全部置き換えて

Claude: lesson-5.md を確認しました。「英語」が3箇所見つかりました。
        すべて「English」に置き換えます。

        変更を適用してもよろしいですか？ (y/n)

あなた: y

Claude: 変更を適用しました。3箇所を置き換えました。
```

## Claude Codeを終了する

使い終わったら、以下のいずれかで終了します：

- `/exit` と入力
- **`Ctrl + C`** を2回押す

## 注意事項

::: warning 大事なポイント
- Claude Code が提案した変更は、**必ず内容を確認してから** 承認してください
- 変更を承認する前に `y`（はい）か `n`（いいえ）を選べます
- 不安な場合は `n` を選んで、もう一度指示を出し直しましょう
:::
