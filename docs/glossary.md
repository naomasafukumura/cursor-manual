# 用語集

わからない言葉があったら、ここで確認してください。

## アプリ・ツール

| 用語 | 意味 |
|------|------|
| **Cursor（カーソル）** | ファイルを編集するアプリ。この中でClaude Codeを使う |
| **ターミナル** | 文字を打ってパソコンに命令する画面。Cursorの中にある。`Ctrl + `` ` で開く |
| **GitHub（ギットハブ）** | チームのファイルを保管する場所。ネット上の共有フォルダのようなもの |
| **Git（ギット）** | ファイルの変更履歴を記録してくれるツール。裏で自動的に動いている |
| **Claude Code** | ターミナルで使えるAI。日本語で話しかけると作業を全部やってくれる |
| **Vercel（バーセル）** | ウェブサイトを公開するサービス。GitHubにファイルを送ると自動でサイトが更新される |

## よく出てくる言葉

| 用語 | 意味 | 例えると |
|------|------|---------|
| **リポジトリ** | ファイルの置き場所 | チームの共有フォルダ |
| **クローン** | ファイルをダウンロードすること | 共有フォルダを自分のパソコンにコピー |
| **ブランチ** | 自分だけの作業スペース | 下書き用のノート |
| **main ブランチ** | 本番のファイル | 教室の黒板 |
| **コミット** | 変更内容を記録すること | 編集した内容をセーブ |
| **プッシュ** | 変更をGitHubに送ること | 封筒をポストに入れる |
| **プル** | GitHubから最新を取ってくること | 最新の資料をもらう |
| **プルリクエスト（PR）** | 「確認してください」とお願いすること | 先生に答案を提出する |
| **マージ** | 下書きの内容を本番に反映すること | 下書きを黒板に書き写す |
| **コンフリクト** | 2人が同じ場所を同時に編集してぶつかること。Claude に「直して」と言えば解決してくれる |

## ファイルの管理ルール

| 何を | どこで |
|------|--------|
| HTML・コード・Markdown | **GitHub**（チームで共同編集） |
| 画像・動画・音声 | **Google ドライブ**（重いファイル） |
| スプレッドシート | **Google ドライブ** |
