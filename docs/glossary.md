# 用語集

わからない言葉があったら、ここで確認してください。

## アプリ・ツール

| 用語 | 意味 |
|------|------|
| **Cursor（カーソル）** | ファイルを編集するアプリ。WordやGoogleドキュメントのようなもの |
| **ターミナル** | 文字を打ってパソコンに命令する画面。Cursorの中にある |
| **GitHub（ギットハブ）** | ファイルをネット上で共有するサービス。Google ドライブみたいなもの |
| **Git（ギット）** | ファイルの変更履歴を記録してくれるツール。裏で自動的に動いている |
| **Claude Code** | ターミナルで使えるAI。日本語で話しかけると作業を手伝ってくれる |
| **Vercel（バーセル）** | ウェブサイトを公開するサービス。GitHubにファイルを送ると自動でサイトが更新される |

## よく出てくる言葉

| 用語 | 意味 | 例えると |
|------|------|---------|
| **リポジトリ** | ファイルの置き場所 | チームの共有フォルダ |
| **クローン** | ファイルをダウンロードすること | 共有フォルダを自分のパソコンにコピー |
| **ブランチ** | 自分だけの作業スペース | 下書き用のノート |
| **main ブランチ** | 本番のファイル | 教室の黒板 |
| **コミット** | 変更内容を記録すること | 編集した内容をセーブ |
| **プッシュ** | 変更をGitHubに送ること | 封筒をポストに入れる |
| **プル** | GitHubから最新を取ってくること | 最新の資料をもらう |
| **プルリクエスト（PR）** | 「確認してください」とお願いすること | 先生に答案を提出する |
| **マージ** | 作業スペースの内容を本番に反映すること | 下書きを黒板に書き写す |
| **レビュー** | 他の人に変更を確認してもらうこと | 先生に答案をチェックしてもらう |

## その他

| 用語 | 意味 |
|------|------|
| **Markdown（マークダウン）** | テキストに簡単な記号をつけて見出しなどを作る書き方。`.md` ファイルで使う |
| **Organization** | GitHubのチーム機能。Masa-English がこれにあたる |
| **Personal Access Token** | GitHubにログインするための「パスワードの代わり」 |
| **コンフリクト** | 2人が同じ場所を同時に編集してしまったときに起きるぶつかり。困ったら管理者に聞いてください |
