# Cursor マニュアル

このマニュアルは、Masa English のメンバーが **みんなで同じファイルを編集できるようにする** ための手順書です。

## やりたいこと

> 「チームのファイルをみんなで編集して、常に最新の状態を共有したい」

これを実現するために、**Cursor**（ファイルを編集するアプリ）と **GitHub**（ファイルを共有する場所）を使います。

## 進め方

**最初に1回だけやること：**

| ステップ | やること | かかる時間 |
|---------|---------|-----------|
| [1. Cursorを入れる](/01-cursor-install/) | 編集アプリをパソコンに入れる | 5分 |
| [2. GitHubに登録する](/02-github-setup/) | ファイル共有サービスのアカウントを作る | 5分 |
| [3. 自分の名前を設定する](/03-git-setup/) | 「誰が編集したか」がわかるように名前を登録 | 3分 |
| [4. ファイルをダウンロードする](/04-clone/) | チームのファイルを自分のパソコンに持ってくる | 3分 |

**毎回の作業（これだけ覚えればOK）：**

| ステップ | やること | かかる時間 |
|---------|---------|-----------|
| [5. 作業スペースを作る](/05-branch/) | 自分の作業場所を用意する | 1分 |
| [6. ファイルを編集する](/06-edit-save/) | 普通にファイルを編集して保存する | - |
| [7. 変更を送る](/07-commit-push/) | 編集した内容をチームに送る | 2分 |
| [8. 確認してもらう](/08-pull-request/) | 他のメンバーにチェックしてもらう | 2分 |
