# Cursor マニュアル

Masa English チームメンバー向けの、**Cursor** と **GitHub** を使ったファイル共同編集ガイドです。

## 対象者

- パソコンの基本操作ができる方
- プログラミングやGitの経験は **不要** です
- Mac / Windows どちらでもOK

## 全体の流れ

| ステップ | やること | 頻度 |
|---------|---------|------|
| [1. Cursorインストール](/01-cursor-install/) | Cursorをダウンロード・インストール | **最初に1回だけ** |
| [2. GitHubセットアップ](/02-github-setup/) | アカウント作成・組織に参加 | **最初に1回だけ** |
| [3. Git初期設定](/03-git-setup/) | 名前・メールアドレスを設定 | **最初に1回だけ** |
| [4. クローン](/04-clone/) | リポジトリをダウンロード | **最初に1回だけ** |
| [5. ブランチ作成](/05-branch/) | 作業スペースを作る | **毎回** |
| [6. 編集・保存](/06-edit-save/) | ファイルを編集して保存 | **毎回** |
| [7. コミット & プッシュ](/07-commit-push/) | 変更をGitHubにアップロード | **毎回** |
| [8. プルリクエスト](/08-pull-request/) | レビューを依頼・反映 | **毎回** |
| [9. Claude Code](/09-claude-code/) | AIアシスタントを活用 | **必要に応じて** |

::: tip はじめての方へ
上から順番に進めてください。ステップ1〜4は最初に1回だけ行う設定です。
ステップ5〜8が毎日の作業フローになります。
:::
