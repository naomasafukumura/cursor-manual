# Cursor マニュアル

Masa English のメンバーが、**チームのファイルをみんなで安全に編集できるようにする** ためのガイドです。

## 仕組み（ざっくり）

```
GitHub（ファイルの保管場所）
    ↕ 同期
自分のパソコン（Cursorで編集）
    ↓ 自動
Vercel（サイトとして公開）
```

- **GitHub** = チームの共有フォルダ（ネット上）
- **Cursor** = ファイルを編集するアプリ（パソコン上）
- **Claude Code** = Cursorの中で使えるAI。日本語で指示するだけで全部やってくれる

## 最初に1回だけやること

| ステップ | やること | 時間 |
|---------|---------|------|
| [1. Cursorを入れる](/01-cursor-install/) | 編集アプリをインストール | 5分 |
| [2. GitHubに登録する](/02-github-setup/) | アカウントを作る | 5分 |
| [3. Claude Codeを入れる](/03-claude-code/) | AIアシスタントをインストール＆ログイン | 5分 |
| [4. 管理者からの招待を受ける](/04-invite/) | チームに参加する | 2分 |
| [5. ファイルをダウンロードする](/05-clone/) | チームのファイルを自分のパソコンに持ってくる | 3分 |

## 毎回の作業（Claude Codeに話しかけるだけ）

| やりたいこと | Claude Codeに言うこと |
|-------------|---------------------|
| 最新にする | 「更新ある？あったら最新にして」 |
| ファイルを編集 | 「このファイルのここを直して」 |
| 変更を送る | 「コミットしてプッシュして」 |
| レビュー依頼 | 「プルリクエスト作って」 |
